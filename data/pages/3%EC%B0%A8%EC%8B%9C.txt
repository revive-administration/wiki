====== 3차시 ======
==== 인클로저 운동 ====
  * 토지를 통해 수익을 확보하는 데 눈을 뜸
  * 농민을 보호해야 했던 전통적 의무에서 벗어남
  * 뉴타운 사업(외부유입 - 지주, 건설사) -> 현대적 인클로저 운동
    * 이는 자본주의의 고유한 성격을 반영한다.
    * 낙후한 외곽에 자본이 재진입하는 현상
      * 좀 더 주변으로 밀려나간다.
  * 1830년대 영국
    * 아동노동

==== 토지와 노동력의 상품화 ====
    * 자급자족적 농업 -> 상업적 농업 (토지가 상품으로 변모됨, 소유권)
    * 예속된 농민 -> '해방된' 프롤레타리아
      * 토지와 농민의 분리, 노동력의 상품화
      * -> 농업에서의 자율성상실
      * 노동력만 제공 -> 농업자본가들의 결정에 따라야 한다.

==== 노동력의 상품화 ====
  * 과거에도 노예는 매매 가능했다.
  * 사람을 파는 것이 아니라 일을 할 수 있는 능력을 파는 것(노동력의 가격)
  * 불안 : 패자부활전이 없다는 것은 20대로 하여금 보수적인 선택을 강요하게 한다.
  * SPEC에 따라 암묵적으로 능력이 결정된다.
    * -> 더 높은 performance를 낼 것이라는 기대감

==== 기계파괴운동 (Luddite Movement) ====
==== 산업혁명의 의미 ====
  * 0 ~ 1820년 - 세계 1인당 연평균 GDP 성장률 0.02 %
  * 1820 ~ 2003년 - 세계 1인당 연평균 GDP 성장류 1.25 %
  * 1820년 => 산업혁명이 본격적으로 시작한 시기

===== 영국 산업혁명의 원인 논쟁 =====
==== 1) 제도적 해석 ====
  * '소유권'을 보장할 수 있는 환경 구축 (제도적 기반)
    * <-> 프랑스 절대주의 : 소수의 부르주아에게만 독점권 제공 (-> 산업혁명 지체)
  * 시장의 활성화 
    * 과거 : 2중적 시장 (원거리 무역(귀중품) / 국지적 시장(생필품)) 
    * -> 전국적 시장망 형성 (거래비용 축소가 있었기에 가능하였음)
  * 제도적 해석에 대한 비판 <자의적이다!>
    * 소유권의 보장은 네덜란드 > 영국이었다.
      * 영구의 명예혁명은 네덜란드의 제도와 관념을 수입하는 과정이었다.
    * 18세기 중국에서는 고도로 발전된 시장이 이미 존재하였다.
    * 명예혁명 이후의 '소유권 보장'은 과장되었다.
      * => 제도적 해석은 기능과 원인을 혼동하는 것으로 보이며 그렇게까지 결정적인 요인은 아니다.

==== 2) 문화적 해석 ====
  * 일종의 민족성(자질, 창의성, 진취성...)
  * 비판
    * 문화적 우위는 뚸렷하지 않았으며, 중국/프랑스보다 기술적으로 아래였음

==== 3) 경제적 해석 ====
  * 산업혁명과 같은 기술 혁신의 '수요'에 주목, 즉 기술혁신의 '수익성'이 가장 중요함
  * 영국경제 특유의 고임금과 저가에너지가 노동절약적인 동시에 자본-에너지 소비적인 '편향적' 기술 진보로의 인센티브를 제공함


==== 기술 진보의 두 가지 양상 ====
  * 1. 산업 혁명 (industrial revolution)
  * 2. 근면 혁명 (industrious revolution)

===== <2강> 경제학의 확립과 전환, 고전파와 신고전파 경제학 / 정치경제학에서 경제학으로 =====
==== 정치경제학(political economy)의 출현 ====
  * 의미전환 : 사적 -> 공적
  * 중상주의(mercantilism)를 배경으로 함
  * 민족국가의 경제적 문제가 주요한 대상이 됨
  * 경제 현상과 경제 정책이 구분되지 않음
  * 법칙적 발견보다는 정책적 실용성에 치우침
  * 상업과 조세를 통한 경험적 자료의 수집을 중시함

==== 중상주의의 전제 ====
  * 다른 나라보다 우위를 갖고 경제 정책을 운용한다 (많은 화폐, 많은 수출)
  * 수입 작게 => 무역흑자 확보
  * 보호 무역 / 식민지 무역(값싼 원제품 수입, 비싼 공업 제품 판매)

==== 중상주의에 대한 비판으로서 중농주의 등장 ====
  * 중상주의 : 구체적, 귀납적 지식 (상업자본의 이해 반영)
  * 중농주의 : 추상적, 연역적 지식 (농업자본의 이해 반영)
  * 사회에도 자연과 마찬가지로 법칙이 작용한다는 가정 수용
  * <phisiocracy> economy = self-regulatin machine (현대 경제학의 근간을 맥락으로 담고 있음)
  * 봉건적 제약으로부터 농업의 해방, 조세체계의 개혁, 농산품 무역 자유화
  * 고전파 경제학의 등장
  * A.Smith<도덕 감정론> : 사회 질서라는 것이 어떻게 가능한가?
    * => 인간의 공감 능력
    * 국부론의 이기심 : 공감능력을 전제로 함
