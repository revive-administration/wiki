====== 스테이터스 재계산 ======
=====정의 =====
스테이터스 재계산은 장비변경, 버프추가 등의 이슈로 인해 재계산이 필요할때 수행한다.

===== 재계산이 필요한 시점 =====
  * 전투시작시
  * 연구를 완료했을 때
  * 장비를 교체할 때
  * 함교에 들어갈때
  * 일반공격을 수행하기 전
  * 오토스킬을 사용하기 전

===== 재계산시 고려해야할 요소 =====
  * 함교 : 아이템 + 특성연구 + 바디패시브
  * 전투 : 아이템 + 특성연구 + 바디패시브 + 버프(항상)
    * 일반공격시 : 아이템 + 특성연구 + 바디패시브 + 버프(항상) + 버프(일반공격시)
    * 스킬사용시 : 아이템 + 특성연구 + 바디패시브 + 버프(항상) + 버프(스킬사용시)
