====== 4장. 머신러닝 ======
===== 4-1. 머신러닝이란? =====
==== 머신러닝 개요 ====
머신러닝은 인간의 뇌가 자연스럽게 수행하는 학습이라는 능력을 컴퓨터로 구현하는 방법이다.

==== 어떻게 특징과 규칙을 찾을까? ====
어떤 요소가 모여있는 것을 "특징량"이라고 부른다. 특징량을 기반으로 구분선을 그으면 컴퓨터도 쉽게 판단할 수 있다. 머신러닝은 이처럼 계산을 통해 구분량을 찾아내는 것이다. 

==== 특징 추출 ====
머신러닝을 하려면 데이터가 어떤 특징을 가지고 있는지 찾고 벡터로 만들어야 한다. 이 같은 변환 처리를 "특징 추출"이라고 한다. 또한 머신러닝에서는 학습을 기반으로 분류 등을 하는 학습 시스템을 "학습 기계"라고 부르는데 "분류/식별기(classifier)", "학습기(learner)"라고 부르는 경우도 있다.

==== 회귀 분석이란? ====
"회귀분석(regression analysis)"이란 통계 용어인데, Y가 연속된 값일 때 Y = f(x)와 같은 모델로 나타내는 것이다. 회귀에서 가장 기본적인 모델은 Y = aX + b이다. 이를 "선형 회귀"라고 부른다. 또한 이러한 모델에서 Y를 연속 측정의 종속 변수(목표 변수), X를 독립 변수(설명 변수)라고 부른다. X가 1차원이라면 "단순 회귀"라고 부르며, 2차원 이상이면 "다중 회귀"라고 부른다.

==== 머신러닝의 종류 ====
^종류 ^ 설명 ^
| 교사 학습 | 데이터와 함께 답을 입력한다|
| ::: | 다른 데이터의 답을 예측한다 |
| 비교사 학습 | 데이터는 입력하지만 답은 입력하지 않는다 |
| ::: | 다른 데이터의 규칙성을 찾는다 |
| 강화 학습 | 부분적으로 답을 입력한다 |
| ::: | 데이터를 기반으로 최적의 답을 찾아낸다 |

===== 4-2. 머신러닝 첫걸음 =====
  * [1] 학습 기계에 데이터를 학습시킨다 - fit() 메서드
  * [2] 데이터를 넣어 예측시킨다 - predict() 메서드
