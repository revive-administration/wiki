====== 성공적인 웹 프로그래밍 PHP와 MySQL ======
=====이 책을 시작하며=====
====PHP란?====
웹에서 설계된 서버 스크립트 언어. PHP 개발은 오픈 소스 프로젝트이다. 정식 버전은 5.0이며 Zend 사의 새로운 엔진에서 완전히 새로 작성되었다.

====MySql이란?====
MySQL은 매우 빠르고 견고한 관계형 데이터베이스 관리시스템(RDBMS)이다. 사용자가 효율적으로 데이터를 저장하고, 검색하고, 정렬하며, 탐색할 수 있도록 도와준다. MySQL은 다중 사용자, 다중 쓰레드 지원 서버이다. MySQL은 표준 데이터베이스 질의 언어인 SQL(Structured Query Language)을 사용한다.

====왜 PHP와 MySQL을 사용하는가?====
PHP의 장점
  * 뛰어난 성능
  * 확장성
  * 다양한 데이터베이스 지원 인터페이스
  * 일반적인 웹 기능을 지원하는 다양한 내장 라이브러리
  * 저렴한 비용
  * 강력한 객체 지향 지원
  * 배우기 쉽고 사용하기 쉬운 문법
  * 호환성
  * 개발 방식의 유연성
  * 소스 코드 공개
  * 충분한 지원과 문서
